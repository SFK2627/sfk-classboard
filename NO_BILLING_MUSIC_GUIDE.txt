No-Billing Music Guide

Firebase Storage requires billing, so this no-billing ClassBoard version avoids Storage.

Best free option:
1. Deploy the updated Code.gs to Apps Script.
2. In Google Drive, create a folder named SFK ClassBoard Music.
3. Upload MP3/M4A files there.
4. In the Memories post page, choose a saved song from the dropdown.

Alternative:
Upload MP3/M4A files manually to your GitHub Pages repo and paste the public direct link.

Notes:
- Google Drive audio links use the Apps Script audio helper. Files must be accessible to the Apps Script account.
- YouTube links cannot be used as hidden background music. Put YouTube links in Video Link instead.
- Direct upload inside Memories is handled by Apps Script/Drive, then Firebase stores only links.
